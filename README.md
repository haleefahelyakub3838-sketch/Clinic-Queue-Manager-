# COS 202 - Health Clinic Queue Manager

This Flask web app manages patient queue for a health clinic (FIFO order).


## What it does
- Register patients (OOP Patient class)
- View waiting list + timestamp
- Call next patient (Queue)
- Shows patients seen today


## How to run locally (for beginner)
1. Install Python
2. Open terminal in this folder
3. Type: pip install flask
4. Type: python app.py
5. Open browser: http://127.0.0.1:5000
